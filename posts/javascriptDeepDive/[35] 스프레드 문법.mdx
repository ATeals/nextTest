> 이 글은 **모던 자바스크립트 Deep Dive**를 읽고 정리한 글입니다.

ES6에 도입된 스프레드 문법은 하나로 뭉쳐 있는 값을 펼쳐서 개별값으로 사용할 수 있게 해줍니다.

스프레드 문법을 사용할 수 있는 대상은 다음과 같습니다.

-   Array
-   String
-   Map
-   Set
-   DOM 컬렉션 (NodeList, HTMLCollection)
-   arguments

> 위 값들은 for … of 문으로 순회할 수 잇는 이터러블 입니다.

주의할 점은 스프레드 문법의 결과는 값이 아니라는 점입니다.

스프레드 문법(…)이 피연산자를 연산하여 값을 생성하는 연산자가 아니기 때문에 스프레드 문법의 결과는 변수에 할당할 수 없습니다.

```jsx
const list = ...[1,2,3] ; // SyntaxError
```

# 함수 호출문의 인수 목록에서 사용하는 경우

---

스프레드 문법을 이용하면 다음과 같이 코드를 작성할 수 있습니다.

```jsx
Math.max(...[1, 2, 3]); // 3
```

# 배열 리터럴 내부에서 사용하는 경우

---

스프레드 문법을 배열안에서 사용하면 기존 방식보다 더 쉽게 사용할 수 있습니다.

## 배열을 합치는 경우

---

기존에 배열을 합치기 위해서는 concat과 같은 메서드를 이용 했습니다.

```jsx
[1, 2].concat([3, 4]);
```

하지만 스프레드 문법을 이용하면 별도의 메서드 없이 결합할 수 있죠

```jsx
[...[1, 2], ...[3, 4]];
```

## 배열 안의 요소를 자르고 값 추가

---

splic메서드를 이용하여 배열 중간에 값을 추가하거나 제거하려면 splice메서드를 사용합니다.

```jsx
[1, 4].splice(1, 0, [2, 3]); // [1, [2, 3], 4]
```

하지만 우리가 기대한 결과값은 [1,2,3,4]이다 이때 스프레드 문법을 이용하면 해결!

```jsx
[1, 4].splice(1, 0, ...[2, 3]); // [1, 2, 3, 4]
```

## 배열 복사하기

---

스프레드 문법을 이용하면 쉽게 배열을 복사할 수 있습니다.

```jsx
const arr = [1, 2, 3];
const arr2 = [...arr];
```

이때 이터러블이 아닌 유사배열 객체는 스프레드 분법을 사용할 수 없습니다.
